# m5s.chat

Project of Theo's cloneathon of [T3.chat](https://t3.chat).

## Features

- [x] - Multi-model and Auto Mode (AI will choose the best model to you).
- [x] - Synchronization
- [x] - Retry messages (and history)
- [x] - Chat branching
- [x] - Syntax highlighting
- [x] - Markdown support
